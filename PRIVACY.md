# Privacy Policy for reqit Interceptor

Last updated: July 2026

The reqit Interceptor Chrome extension does not collect, transmit, or share any personal data, browsing history, or identifiable information.

All HTTP/S request data captured by this extension is forwarded exclusively to a local proxy server running on the user's own machine (127.0.0.1:3100 by default). No data ever leaves the user's device.

Only connection settings (proxy host and port) are stored locally via chrome.storage.local. No browsing data is persisted or transmitted.

This extension has no affiliation with any third-party data collectors, analytics services, or advertising networks.

For questions, open an issue at: https://github.com/HalxDocs/reqit/issues
